## Como Usar

1. Baixe o arquivo `.json` da paleta desejada na pasta `Arquivos JSON/`
2. No Power BI Desktop, va em **Exibir > Temas > Procurar temas**
3. Selecione o arquivo `.json` baixado
4. O tema sera aplicado automaticamente ao seu relatorio

## Versoes Acessiveis

Cada paleta possui uma versao `_Acessivel` otimizada para:
- Daltonismo (protanopia, deuteranopia, tritanopia)
- Baixa visao
- Contraste adequado segundo WCAG 2.1
